VO-DML Documentation Sources
============================

The subdirectories contain the sources for the various kinds of documentation related to VO-DML

* [std](./std/) contains the LaTeX source for the formal IVOA VO-DML standard that is published [on the IVOA website](https://www.ivoa.net/documents/VODML/)
* [guide](./guide/) contains a mkdocs site that describes the [vo-dml tooling](../tools) that is the main subject of this repository.
* [note](./note/) contains the LaTeX source of a document that describes the development of the tooling.
